^^ LAB - `Compare01`


## DESCRIPTION

Create a program that prompts the user for an integer. The program should compare the user's number to 10, and then print an appropriate message. The program must end with a message that indicates that the program is done running.

^=^ ,, 10

**NOTE: Assume the user enters a valid integer.**

^=

## SPECIFICATIONS

* Starter code has been provided for this lab.

* File name: `Compare01.java`

* The code to complete the task should be in a separate method that is called by the `main` method

* Add comments with your name, date, and program description at the top!


## SAMPLE OUTPUT

```
Enter an integer:
14
The number 14 is GREATER than 10.
Program finished!
```

```
Enter an integer:
9
The number 9 is NOT GREATER than 10.
Program finished!
```

```
Enter an integer:
10
The number 10 is NOT GREATER than 10.
Program finished!
```
