^^ LAB - `Compare02`


## DESCRIPTION

Create a program that prompts the user for two integers. The program should compare the first number to the second one and then print an appropriate message. The program must end with a message that indicates that the program is done running.

^=^ ,, 10

**NOTE: Assume the user enters valid integers.**

^=

* Starter code has been provided for this lab.

* File name: `Compare02.java`

* The code to complete the task should be in a separate method that is called by the `main` method

* Add comments with your name, date, and program description at the top!

* Output should match the Sample Output exactly.
