"Backend" is the term in web development that means "what happens on the *server*... That is, some computational work that happens *before the content leaves the server and the user sees it*. Once the backend server does it's thing (that might mean peaking into other files, it might mean changing some content, and it might mean creating new files), *then* the server sends it to the user.
