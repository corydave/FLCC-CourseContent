^^ LAB - rectangle [Description]


## DESCRIPTION

Create a script that displays the a rectangle on four lines. The first and fourth line should be five asterisks and the second and third line should consist of one asterisk, three spaces, and another asterisk.

## SPECIFICATIONS

* File name: `rectangle.py`
* Be sure to modify the comments
* Be sure to output the exact message as shown in the sample output

## SAMPLE OUTPUT

```
*****
*   *
*   *
*****
```
