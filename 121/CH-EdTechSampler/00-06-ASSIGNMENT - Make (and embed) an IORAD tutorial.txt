^^ ASSIGNMENT - Make (and embed) an IORAD tutorial

^= =

[IORAD](https://www.iorad.com/) is an interesting piece of software; it allows you to make step-by-step tutorials. Once you create it, you can distribute it as an interactive tutorial (where the participant has to walk through the process by clicking on buttons), a PDF instruction manual, a video, a step-by-step list, or as a quiz. 

It's quick to learn, but you need to spend some time if you want to master it (which you do not need to for this class). You will, however, be making a tutorial using IORAD for this assignment. It does NOT need to be a tutorial that covers a complicated topic; it can be pretty simple (like how to log into a website like FlipGrid, or how parents can access grades in a class, or how to do something in Google Docs, for example). 

It should:
* **Be easy to understand from the text:** you might have to tweak the words that are automatically generated

* **Be accurate:** the hit-boxes must be accurate and the steps/flow of the tutorial should lead to accurate completion

* **Be presentable:** the final product should be good-to-go for a class you would be teaching

You will have to:
* Install the Chrome Extension (or Firefox, or Edge).

* Watch this tutorial video - note that the tutorial goes over some advanced maneuvers (the dragging and dropping, for instance). You do not need to include the more difficult moves in your tutorial.
