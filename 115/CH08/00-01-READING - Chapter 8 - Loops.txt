^^ READING - Chapter 8 - Loops

The reading for this chapter is [Chapter 8 - Loops](https://docs.google.com/document/d/1HAPpbdoJkwmobvSKDeNsoJD_9xki7YR7NdEBFJzfojs/). We're about to look at *loops* - that is, code that repeats until a certain condition is met. There are a few different flavors of loops in Java. Let's take a brief look at them. You **do not** need to understand what the following code segments do (yet), but hopefully you can start forming some predictions:

```java

String choice = "yes";

while (choice.equals("yes") {
  System.out.print("Play again?");
  choice = scanner.nextLine();
}
```

<br />

```java

for (int i = 0; i < 10; i++) {
  System.out.println(i);
}

```
