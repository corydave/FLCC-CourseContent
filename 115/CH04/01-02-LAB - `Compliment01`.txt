^^ LAB - `Compliment01`

## DESCRIPTION

This is a simple program that will dole out a compliment to the user:

1. Ask the user their age
2. Ask the user their name
3. Output a compliment based on their information


## SPECIFICATIONS

* Starter code has been provided for this lab.

* File name: `Compliment01.java`

* Your output should match the sample output exactly.

* The code to complete the task should be in a separate method that is called by the `main` method

* Add comments with your name, date, and program description at the top!

* All output statements use `println()`, not `print()`.
