# Code from class

```html

<!DOCTYPE html>
<!-- A quick intro to adding some JavaScript -->
<html>
 <body>
  <h1>JavaScript Basics</h1>
  <p id="random">this is not random</p>

  <!-- You can insert JavaScript directly into -->
  <!-- an HTML file with the script element:   -->
  <!-- between opening and closing script tags -->

  <script>
    //css style selection to "grab" element off page
    let p = document.querySelector("#random");
    
    //generate a random number (rng time!)
    let num = Math.floor(Math.random()*10); //0 to 9
    
    //change the innerHTML of the element we grabbed
    p.innerHTML = "Random Number: " + num;
    
    //change the style! Notice, we build the CSS
    p.style.fontSize = num * 4 + "pt";
    p.style.backgroundColor = "#5B" + num;
  </script>

 </body>
</html>

```
