Snippet #2

last_name = "Hopper"
print(first_name + " " + last_name)
Reveal Answer

Output:

Grace Hopper

Explanation: This is called string concatenation. Don't forget the space in the middle!

Snippet #3

x = 10
x = x + 5
print(x)
Reveal Answer

Output:

15

Explanation: Variables can be updated. We take the current value of x (10), add 5, and assign it back to x.
