^^ LAB - `file_writer_reader`

## DESCRIPTION
This lab is intended to explore file writing capabilities with Python. In this lab, there are three different files on the server - `pokemon.txt`, `marvel.txt`, and `movies.txt`. Your job is to:
1. Ask the user for the name of a file
2. Ask them for an entry
3. Add the entry to the end of the file
4. Close the file
5. Open the file and print out the contents

## SPECIFICATIONS

* Starter code has been provided for this lab.

* Add a comment with your name, date, and program description at the top!

* Your output must match the Sample Output *exactly*.
