^^ LAB - `random_number`


## DESCRIPTION

Write a program that outputs a random number using the `random` module in Python.

## SPECIFICATIONS

* Starter code has been provided for this lab.

* The filename for this program is `random_number.py`

* Complete the `generate_random_number` function. The sole purpose of this function is to output a random number

* Add comments with your name, date, and program description at the top.

* The very first line of this program should be `import random`; if you do not include it you may have access to the `random` module.

* The auto-test for this program will only check to see that a random number has been output; it will not check to see that the random number is any specific number.

## SAMPLE OUTPUT

```
0.8525041884365221
```

```
0.13938116419681712
```

```
0.6717733648463142
```
