üé≤ The Random Module

Purpose: The random module allows programs to simulate chance. This is essential for games (rolling dice), simulations (weather patterns), and security (cryptography).

Sample Program: Virtual Dice

This program simulates rolling a 6-sided die.

import random

roll = random.randint(1, 6)
print("You rolled a " + str(roll))

if roll == 6:
    print("Critical Hit!")
elif roll == 1:
    print("Critical Miss!")
else:
    print("Nice roll.")

üêû Find the Error

This program tries to ask the user for a number and check if it is 2. There are three errors. Can you find them?

user_choice = input("Enter a number: ")

if user_choice = 2
    print("You chose two!")
else:
    print("You chose something else.")
Reveal Answer
  1. Type Mismatch: input() returns a string. It should be int(input(...)) to compare with the number 2.
  2. Assignment vs Equality: The if statement uses = (assignment) instead of == (comparison).
  3. Syntax Error: The if statement is missing the colon : at the end of the line.
