^^ LAB - `glue_name`

## DESCRIPTION

Write a program that will ask the user for their first name and their last name. Then output the word "Hi," followed by their full name.

## SPECIFICATIONS

* Starter code has been provided for this lab.

* The filename for this program is `glue_name.py`

* Complete the method `glue_name()`

* Add a comment with your name, date, and program description at the top!


## SAMPLE OUTPUT


```
Dave
Ghidiu
Hi, Dave Ghidiu
```

```
bart
simpson
Hi, bart simpson
```

```
Lisa
Simpson, esquire
Hi, Lisa Simpson, esquire
```
