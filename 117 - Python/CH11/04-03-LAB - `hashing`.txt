^^ LAB - `hashing`

## DESCRIPTION
In this program, you are tasked with using the `sha256` function in `hashlib` to hash input from the user.

## SPECIFICATIONS

* Starter code has been provided for this lab.

* Add a comment with your name, date, and program description at the top.
