^^ LAB - `compliment`

## DESCRIPTION
In this program, the user is prompted whether they want a compliment or not. If they choose "yes", then a compliment is given. 

Either way, the user is thanked for participating.

## SPECIFICATIONS

* Starter code has been provided for this lab.

* Add a comment with your name, date, and program description at the top!

* Note that *any input other than* `'yes'` should not be considered an affirmative answer. That includes capitalized versions.


## SAMPLE OUTPUT


```

You have wonderful eyes.
Thank you for playing.
```
